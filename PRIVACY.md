# Privacy Policy

Last updated: April 16, 2026

Decision Simulator is a Claude Code plugin distributed through GitHub.

## What this plugin does

Decision Simulator adds local plugin files for Claude Code, including skills and an agent definition, to improve technical decision-making during Claude Code workflows.

## Data collection

Decision Simulator does not operate its own hosted service.

Decision Simulator does not independently collect, store, or transmit user prompts, files, code, outputs, or personal data to a server controlled by the plugin author.

## How the plugin runs

The plugin runs through Claude Code using local plugin files contained in this repository.

This repository does not include:

- a hosted backend
- analytics or telemetry code
- external webhooks
- plugin-managed remote APIs
- an MCP server

## Third-party processing

When you use this plugin, your data may still be processed by third-party services you choose to use, including:

- Anthropic, when you use Claude Code or submit the plugin through Anthropic surfaces
- GitHub, when you access the repository or install from GitHub

Those services operate under their own terms and privacy policies.

## Data retention

Decision Simulator does not maintain its own backend or database for retaining user prompts, code, outputs, or personal data.

## Changes to this policy

If the plugin later adds a hosted service, telemetry, remote APIs, or other data collection behavior, this privacy policy will be updated to reflect those changes.
